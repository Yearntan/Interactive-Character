Cute spongebob

There are three buttons, happy, sad and neutral which indicates different face expressions.

When user clicks on sad button, eyebrows, cheeks and mouth of cartoon change

When user clicks on happy button, eyebrows, cheeks and mouth change and teeth appear.

Eyes for sad face and happy face are the same.

When user clicks on neutral button, again, eyebrows, cheeks, eyes and lips changes

Beard appears when user clicks on chin area, and it will disappear when users click on chin again.

Details of beard will be different for each expression. For instance, the colour will be white for neutral face, yellow for happy face and blue for sad face.

New hairstyle when user clicks on hair area, and it will go back to the original hairstyle when users click again.

Details of new hairstyle will be different for each expression. For instance, the colour will be white for neutral face, yellow for happy face and blue for sad face.

This canvas is mobile-first responsive, meta vieport is added.

For example, font-size is 16px when screen size is below 600px so that fonts are clearly visible

For desktop, font-size is 24px.
